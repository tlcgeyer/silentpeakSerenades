Tables containing HTTP methods and Data

A table for the landing page

HTTP Method Path Description
GET "/" or "/silentpeakSerenades" Landing page

A table for the User data

HTTP method Path Description
GET "/users" To fetch all Users
GET "/users/:id" To fetch a single user
POST "/users/addUser" To add a new user
PATCH "/users/updateUser/:id" To edit user details
DELETE "/users/deleteUser/:id" To remove a user
POST "/login" To log a user into the application

A table for the Product data

HTTP method Path Description
GET "/products" To fetch all products
GET "/products/:id" To fetch a single product
POST "/products/addProduct" To add a new product
PATCH "/products/updateProduct/:id" To edit product details
DELETE "/products/deleteProduct/:id" To remove a product
